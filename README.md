# 未来地域支援機構 公式LP

このフォルダ一式を GitHub にアップロードすると、GitHub Pages で公式サイトとして公開できます。

## 公開手順

1. GitHubでアカウントを作成
2. 新しい公開リポジトリを作成（例: `mirai-support-org`）
3. このフォルダ内のファイルをすべてリポジトリ直下にアップロード  
   - `index.html`
   - `assets/style.css`
   - `assets/logo.png`
   - `assets/stamp.png`（任意）
   - `.nojekyll`
4. リポジトリの **Settings → Pages** を開く
5. 「Source」で `main`ブランチ / `root` を選択して保存
6. 数十秒〜1分後、表示されたURLにアクセス  
   例: `https://ユーザー名.github.io/mirai-support-org/`
